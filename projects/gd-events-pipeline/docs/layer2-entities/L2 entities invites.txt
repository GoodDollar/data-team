Invitees (1 invitee per row)
	Invitee Address
	Inviter Address
	Signup type (referral/campaign)
	Signup timestamp
	1st claim timestamp
	Total claim count
	Bounty tx hash

Inviter_relationship (one relationship per row)
	Invitee Address
	Inviter Address
	Signup TX hash
	Signup timestamp
	Bounty TX hash
	Bounty timestamp

Invite_Signups (one signup event per row)
	Signup TX hash
	Signup TX timestamp
	Signup type (inviter/referral/campaign)
	User Address
	
Invite_Payouts (one bounty payout event per row)
	payout TX hash
	payout TX timestamp
	Bounty type (referral/campaign)
	Invitee Address
	Inviter Address
	Invitee reward amount
	Inviter reward amount
	Campaign returned amount
	Total amount (either G$1500 when a referral bounty or G$500 when a campaign bounty)
